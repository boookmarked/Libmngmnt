Library Management System (MySQL Based)

A Python-based digital library system that uses MySQL database for storing and managing data.

Features
Admin & User authentication
Add, update, delete books
Issue & return system with fine calculation
User management
Notes system
Search functionality (books & users)
Tech Stack
Python
MySQL
mysql-connector
How to Run

Install MySQL and create a database named:

library

Install required library:

pip install mysql-connector-python

Update database credentials in code:

host="localhost", user="root", password="your_password"

Run the program:

python main.py
Note

Make sure MySQL server is running before executing the program.
